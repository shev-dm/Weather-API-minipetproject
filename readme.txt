Структура проекта:

Weather-API-petproject/
├── my-city/
│    └──my-city.go
│── weather/
│    └──weather.go
│── main.go


Реализация:
Веб-сервер, реализующий получение погоды от сервиса OpenWeatherMap API (https://openweathermap.org/api). Для получения погоды нужно передать 2 параметра my_city и city. 
my_city должно быть равно 0 или 1. При my_city=1 не учитывается параметр city и погода опрделяется в городе где запущен данный сервер. Город определяется с помощью сервиса http://ip-api.com
При значении my_city!=1 считывается значение city и по этому городу возвращаются данные о погоде. 
В обоих случаяш данные возвращаются в формате json
Пример вызова: 
http://localhost:8080/weather?my_city=0&city=Moscow
http://localhost:8080/weather?my_city=1

Запуск:
Для запуска выполнить команду go run main.go или скомпилировать каждую версию в исполняемый файл с помощью go build, который затем можно будет запускать напрямую.

Обратите внимание, что для правильной работы пакета weather.go необходимо получить ключ API на сайте OpenWeatherMap API ( https://openweathermap.org/api ) и ввести его при запуске сервера.
